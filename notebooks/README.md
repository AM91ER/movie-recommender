# Notebooks

This directory contains the Jupyter notebooks used for data exploration and model development.

## Notebooks

| Notebook | Description |
|----------|-------------|
| `01_Data_Understanding_Cleaning_Sampling.ipynb` | Data loading, cleaning, and stratified sampling |
| `02_EDA.ipynb` | Exploratory Data Analysis with visualizations |
| `03_ML_Preprocessing.ipynb` | Feature engineering and train/val/test splits |
| `04_Model_Training.ipynb` | Model training and evaluation |

## Running the Notebooks

1. Install Jupyter: `pip install jupyter`
2. Start Jupyter: `jupyter notebook`
3. Run notebooks in order (01 → 02 → 03 → 04)

## Data Requirements

Before running notebooks, download the MovieLens 32M dataset:
- Source: https://grouplens.org/datasets/movielens/32m/
- Extract to `../data/raw/`
